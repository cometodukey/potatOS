# potatOS

potatOS is a small operating system for i686.

Documentation can be found [here](docs/README.md).

## Features

* Multiboot 0.6.96
* An almost standards compliant printf
* Stack trace (with symbols!)
* Serial
* 8259 PIC
* VGA text mode console
* Physical memory allocator
